## 결정 1: 홈 화면 정보 계층

```
┌─────────────────────────────┐
│ [절기색 배경] 사순절 23일째 · 3월 25일  │ ← FIRST: 절기 배너
├─────────────────────────────┤
│  "주께서 나의 목자시니         │
│   내게 부족함이 없으리로다"   │ ← SECOND: 말씀 본문 (Hero)
│              — 시편 23:1    │
│  [묵상 가이드 한 줄...]        │ ← THIRD: Secondary
├─────────────────────────────┤
│       ✓ 오늘 읽기 완료       │ ← CTA: 하단 고정
└─────────────────────────────┘
```

**탭 바 구성:** 홈 | 소그룹 | 진도 | 설정 (하단 4탭)

**화면별 계층:**
- 소그룹 피드: 오늘의 말씀 공유 배너 → 멤버 나눔 피드 → 하단 입력창
- 나의 진도: 스트릭 카운터 → 잔디 그래프 → 일독 진도율 바

## 결정 2: 인터랙션 상태 (전체)

### 소그룹 빈 화면 (코드 미입력)

```
┌─────────────────────────────┐
│    👥  같이 읽고 나누면       │
│       더 오래 계속됩니다      │
│                             │
│  [코드 만들기]               │ ← Primary CTA
│  [코드 입력하기]             │ ← Secondary CTA
│                             │
│  소그룹 없이도 혼자 사용할    │
│  수 있어요 → [홈 화면으로]   │ ← Escape hatch
└─────────────────────────────┘
```

### 완료 체크 성공 피드백

- 잔디 해당 셀: 절기 색상으로 실시간 채워짐 + 체크마크 바운스 애니메이션
- 전체 화면 모달/confetti 없음 (종교 앱 문맥에 과함)

### 인터랙션 상태 테이블

| 화면 | LOADING | EMPTY | ERROR | SUCCESS |
|------|---------|-------|-------|---------|
| 홈 (오늘의 말씀) | 스켈레톤 UI | — | "불러오기 실패" + 재시도 | 잔디 업데이트 + 체크 애니메이션 |
| 소그룹 (코드 없음) | — | 스토리 + CTA 2개 + Escape hatch | — | — |
| 소그룹 (글 없음) | 스피너 | "첫 번째가 되어보세요 👋" | "피드 실패" + 재시도 | 나눔 글 실시간 표시 |
| 나의 진도 | 스켈레톤 | "오늘 첫 체크를 해보세요!" | — | — |
| SQLite 초기화 | DatabaseInitScreen | — | DatabaseErrorScreen (3회) | 홈 자동 전환 |

## 결정 3: 온보딩 톤

**소그룹 대화 스타일** — 20~30대 교회 친구가 실제로 말하는 어조

- 시작 메시지: "주일만 독실한 신자가 될 필요 없어요, 오늘 하루 2분이면 충분해요"
- 스트릭 0일: "오늘부터 시작해봐요!"
- 스트릭 단절 재시작: "누구나 빠뜨리는 날이 있어요 ✨"
- 소그룹 첫 참여: "처음 오셨네요 👋 편하게 한 줄만 나눠봐요"

## 결정 5: 타이포그래피 시스템

| 용도 | 폰트 |
|------|------|
| UI 전반 (라벨, 온보딩, 버튼) | Pretendard |
| **성경 본문** | **Gowun Batang** (세리프) |
| 스트릭/숫자 강조 | Pretendard Bold |

성경 본문에 세리프를 써서 "읽는 모드"와 "UI 모드"를 시각적으로 구분.

**Spacing scale:** 8 / 12 / 16 / 24 / 32 / 48px (4pt 배수)
**Border radius:** 8px (카드), 16px (버튼), 24px (모달)
**Shadow:** 최소화 — decorative shadow 없음

## 결정 6: 다크 모드 (v1 포함)

**OS 선호 자동 감지** — light/dark 모두 v1에 포함.

구현 시:
- 모든 절기 색상 팔레트에 dark variant 정의 (위 4A 테이블 참조)
- 기본 구현 대비 +1주 예상

**접근성 기준:**
- 최소 터치 타겟: 44pt (iOS HIG)
- Korean VoiceOver 라벨: 완료 체크 버튼, 탭 바 각 탭
- 색상 대비: WCAG AA (4.5:1 이상)
- 성경 본문 컨테이너: 스크롤 가능
